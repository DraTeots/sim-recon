For a parabola:
$$
y = ax^2 + bx + c
$$
The minimum is at
$$
(x,y) = \left( -{b \over 2a}, -{b^2\over2a} + c   \right)
$$
Let $(-1,y_-)$, $(0,y_0)$, and $(1,y_+)$ define a parabola. Then
$$
a = {y_+ - 2y_0 + y_- \over 2}
$$
$$
b = {y_+ - y_- \over 2}
$$
$$
c = y_0
$$
